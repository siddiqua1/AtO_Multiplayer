## Usage

 1. Make a folder in BepInEx\plugins called `cards`
 2. Place json files into BepInEx\plugins\cards\

## JSON template
Minimum for card to show up  
```json
{
	"cardName": "SampleCard",
	"id": "SampleCard",
	"showInTome": true
}
```
